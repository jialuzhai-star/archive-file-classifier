# 文件读取与判定速查

本技能不依赖外部 API。

保留本文件作为快速备忘，记录常见文件类型的只读判定方式：

## PDF

- 先读首页、目录页、前 3-5 页
- 扫描版优先 OCR
- 重点看标题、作者、日期、结论页、章节名

## XMind

- 解压读取 `content.json` 或等效结构文件
- 重点看中心主题、一级二级节点、分支命名

## XLSX

- 先看 sheet 名
- 再看首行字段、样例数据、汇总页
- 区分“原始底表”与“分析成稿”

## DOCX / PPTX

- 先读封面标题、目录页、结论页、备注区
- 判断文档主要交付目的，不按零散章节强行归类

## 归类卡住时

- 不强判
- 放入待确认清单
- 向用户说明候选分类和卡点
